Variables

Las variables son un espacio reservado de memoria que lo usaremos para guardar datos. Puedes pensar que es una caja a la que le ponemos un nombre y la usamos para guardar un valor.

Para definir las variables en js se usan las palabras var, let y const. Para declararlas (asignarles un valor) se usa el signo = .

var nombre = "Pedro" ;

let numero = 4 ;

const PI = 3,14 ;

Javascript es un lenguaje débilmente tipado y dinámico. Esto quiere decir que las variables no están asociadas a un tipo especifico, es por eso que en js cuando definimos una variable no se define el tipo de valor que contiene.

Recuerda

  • Utilizar let, en lugar de var, porque tiene un alcance de bloque y esto nos evitará posibles problemas.
  • Las variables de tipo const no pueden cambiar su valor luego de ser declaradas.
  • Usar nombres descriptivos para las variables. Si estas calculando el area de un triangulo es mejor nombrar a las variables como base, altura en lugar de a, b .
  • Es buena practica definir las variables al principio del archivo js, para que sea más facil leer el codigo.

Consola y Alertas

La consola nos será muy útil para mostrar datos, información, descubrir errores, etc. Puedes usar la consola del Visual Studio Code o la de tu navegador.

Para mostrar datos por consola se usa la sentencia console.log().

let nombre = "Diego";

console.log( 25 ); // 25

console.log( "Hola mundo" ); // Hola mundo

console.log( nombre ); // Diego

Para ingresar datos se usa la sentencia prompt(). Dentro de los paréntesis se agrega el mensaje que se mostrará junto con un input. Recordá que todo lo que ingresa en un prompt es una cadena de texto. Si pedis que se ingresen numeros y luego los vas a usar para realizar operaciones matematicas, primero tenes que usar parseInt() para transformar el el texto en enteros.

let nombre = prompt("Ingrese su nombre");

let nota = parseInt( prompt("Ingrese la nota del primer examen.") );

Por ultimo, si queremos mostrar informacion o un mensaje se usa la sentencia alert(). Muestra una ventana flotante sobre nuestra pagina con un mensaje que queremos mostrar.

alert("Hola mundo");
