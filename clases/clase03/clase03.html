Ciclos

Los ciclos son estructuras que nos permiten repetir operaciones. Estas repeticiones pueden ser una cantidad de veces definidas o pueden estar sujetas a alguna condición.

Estructura For

Este ciclo se repite hasta que una condición se evalúe como False. Su estructura es:

for (ExpresionInicial ; expresionCondicional ; ExpresiónDeActualizacion ) {

// Codigo que se ejecuta si la condición es True

}

Cuando se ejecuta un ciclo For, ocurre:

  • ExpresionInicial: Primero, se ejecuta la expresión inicial, en esta expresión normalmente se define e inicia un contador de bucle. Se suele usar la letra i para definir el contador.
  • ExpresionCondicional: Segundo, se evalúa la expresion condicional. Si el valor de la condición es True, se ejecuta el codigo dentro del ciclo. Si el valor de la condición es False, la ejecución del ciclo termina.
  • ExpresionDeActualizacion: Luego de ejecutar el codigo dentro del ciclo. Se ejecuta la expresión de actualización del contador y vuelve a ejecutarse el ciclo.

Estructura While

El ciclo while nos permite repetir intrucciones siempre que una condición especificada se evalúe como True. Su estructura es:

while( condición ) {

// Codigo que se ejecuta si la condición es True

}

Primero se ejecuta la condicion del while. Si se evalúa como True, se ejecutara el codigo dentro del bucle y se repetirá mientras la condición sea True. Si la condición evaluada resulta False, el ciclo se termina.

  • Luego de ejecutar todo el codigo dentro del ciclo, se vuelve a evaluar la condición.
  • Recordá definir dentro del ciclo un control para finalizar su ejecución. Es muy facil generar un bucle infinito... Muy importante

Estructura Do While

El ciclo Do While nos permite repetir instrucciones de codigo por lo menos una vez y mientras una condición sea True. Es parecio al While, pero este se ejecuta si o si una vez. Su estructura es:

do {

// Codigo que se ejecuta dentro del ciclo

} while( condicion )

Al ejecutarse un Do While, primero se ejecuta el codigo que contiene y luego se evalúa la condición. Si la condición es True, se repite el ciclo. Si la condición es False, se termina el ciclo.

  • Al igual que while, recuerda definir un control de salida del ciclo, para evitar un bucle infinito.

Break y Continue

Existen sentencias que nos permiten alterar la ejecución de los ciclos. Estas son el break y continue

El break es una sentencia que finaliza la ejecución del ciclo que lo contiene. Entonces si lo agregas dentro de un ciclo for, cuando se ejecute el break, el for se termina sin importar si aún podia seguir repitiendose. Es muy útil usarlo cuando se cumple alguna condición prohibida y queremos finalizar un ciclo. Por ejemplo, estamos pidiendo datos al usuario y hay un limite de intentos, cuando el usuario exceda los limites se ejecuta el break y finaliza la petición.

El continue es una sentencia que saltea la repetición del ciclo y sigue con la siguiente. Si lo usas en un ciclo for cuando se ejecute el continue, se finaliza esa repetición e inicia una nueva, con el contador del for actualizado. Un uso practico sería usarlo cuando no queremos tratar algunos casos especificos,

Supongamos que queremos mostrar todos los numeros pares entre 0 y 20. Se utiliza el continue para omitir los numeros impares

for (let i = 0 ; i <= 20 ; i++ ) {

if( i % 2 !== 0 ) { // Si el resto de dividir el valor de i por 2 es distinto de cero, es porque es un numero impar

continue;

}

console.log( `Numero par: ${ i }` );

}

Switch

La estructura switch está diseñada para manejar diferentes condiciones. Tiene una expresion y varios casos definidos que se ejecutaran según el valor de la expresion. La estructura del switch es:

switch ( expresion ) {

case 1 :

//codigo que ejecuta el caso 1

break;

case 2 :

//codigo que ejecuta el caso 2

break;

case 3 :

//codigo que ejecuta el caso 3

break;

default :

//codigo que ejecuta el caso default, si ninguno de los casos coincide con la expresión

break;

}

Cuando se ejecuta el switch, se evalúa la expresión y se la iguala de forma estricta "===" con los diferentes casos definidos. Si la expresión es igual a alguno de los casos, entonces se va a ejecutar el codigo que contiene ese caso, luego se ejecuta la sentencia break que termina el switch. Si ninguno de los cases es igual a la expresión, se ejecuta el caso default.

Un ejemplo de su uso sería cuando mostramos un menu de opciones al usuario y le pedimos que elija una opción. Podemos decirle que escriba el numero o nombre de la opción y usar un switch para ejecutar algo según la opción que ingrese.
