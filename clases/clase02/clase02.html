Condicionales

El codigo que escribimos en js se ejecuta de arriba hacia abajo, podriamos decir que en linea recta. En los lenguajes de programación existen escructuras llamadas CONDICIONALES, estas estructuras nos permiten tomar decisiones en base a una condición que definimos. Con estas estructuras podremos tomar distintos caminos durante la ejecución del codigo.

Para tomar estas decisiones se usan las variables de tipo Booleano. Este tipo de variable solo puede tomar 2 valores: true o false.

let valorValido = false; // False

let mayorDeDiez = ( 5 + 7 ) > 10; // True

La estructura condicional más usada es el if su estructura es:

if (condición) {

//codigo que ejecuta si condición es True

}

Solamente se ejecutará el bloque de codigo dentro de las llaves, si la condición es true.

Tambien se puede usar la extructura if else. Que ejecutará uno y otro bloque de codigo según la condición es True o False.

if (condición) {

//codigo que ejecuta si condición es True

} else {

//codigo que ejecuta si condición es False

}

Operadores

Los operadores en los lenguajes de programación son simbolos que nos permiten realizar distintas operaciones.

Tipos de operadores

Operadores aritmeticos

  • Suma: 3 + 5 // 8. "hola" + "mundo" // "Holamundo".
  • Resta: 5 - 2 // 3.
  • Negativo: -5 // -5.
  • División: 20 / 4 // 5.
  • Modulo Aritmetico: 7 % 4 // 3, me devuelve el resto de la division.
  • Multiplicación: 4 * 4 // 16.
  • Incrementos y Decrementos: ++ , -- .

Operadores de Comparación

Estos operadores nos van a devolver como resultado un valor Booleano

  • Igualdad: == // iguales en valor, pueden ser distintos tipos de datos
  • Igualda Estricta: === // iguales en valor y en tipo de dato
  • Desigualdad: !=
  • Desigualdad Estricta: !==
  • Mayor que: >
  • Mayor o igual que: >=
  • Menor que: <
  • Menor o igual que: <=

Operadores de Asignación

  • Asignación Simple: =

Ademas el operador de asignación puede combinarse con los operadores aritmeticos para obtener los siguientes operadores:

  • a += b. // Equivale: a = a + b
  • a -= b. // Equivale: a = a - b
  • a /= b. // Equivale: a = a / b
  • a *= b. // Equivale: a = a * b
  • a %= b. // Equivale: a = a % b

Operadores de Logicos

Estos operadores van a comparar valores Booleanos y nos van a devolver un valor Booleano

  • Negación Lógica (not): ! // Lo que hace es asignar el valor contrario. Si es True, lo cambia a False y viceversa.
  • Conjunción Lógica (and): && // Solo devuelve true, si ambas expresiones son true. De lo contrario devuelve False.
  • Disyunción Lógica (or): || // Devuelve True si al menos una de las expresiones es True. Devuelve False cuando ambas sean False.
